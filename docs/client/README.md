# Client Documentation

This section provides information about the client-side mods for **Fabric** and **NeoForge**.

## Modules

### Fabric
The `client-fabric` module uses the Fabric API and Loom to integrate with the game.

### NeoForge
The `client-neoforge` module utilizes the NeoForge modding API to provide the same features as the Fabric version.

## Functionality
Both client mods are responsible for detecting player inputs and sending them to the server via custom packets.
