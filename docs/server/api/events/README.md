# Events API

InputEngine exposes custom Bukkit/Spigot events that your plugins can listen to. This allows you to react to player inputs seamlessly.

## Available Events

* [**PlayerKeyPressEvent**](PlayerKeyPressEvent/README.md): Fired when a player presses or releases a registered key, including mouse buttons and scroll wheel.
* [**PlayerComboEvent**](PlayerComboEvent/README.md): Fired when a player completes a sequence of key inputs that match a registered combo.
